# Boatshed concept implementation

This version rebuilds the homepage against the supplied Boatshed concept image rather than the earlier cartoon/directory-style attempts.

## Implemented direction

- Deep navy header and hero frame
- Boatshed wordmark with gold wave accent
- Large photographic yacht hero
- Premium white search panel with tabs
- Broker inspected / survey / finance / video trust strip
- Refined featured boat cards
- Explore by category pills
- Secondary chandlery/services/marina marketplace layer
- Navy/gold trust statistics band
- Responsive mobile layout

## Build check

`npm run build` has been run successfully in this environment with Next.js 14.2.35.
