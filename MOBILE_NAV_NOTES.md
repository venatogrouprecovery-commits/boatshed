# Mobile navigation update

This build adds a proper responsive hamburger navigation for tablet/mobile.

## What changed

- Added a native, accessible `<details>` hamburger menu in `app/layout.tsx`.
- Desktop navigation remains visible on large screens.
- Mobile/tablet navigation switches to a hamburger button.
- The mobile panel includes Boats, Chandlery, Services, Marinas, Sell, Brokers, Sign in/Dashboard and List your boat.
- Styled the menu in `app/globals.css` with premium navy/gold treatment, tap-friendly rows and animated hamburger lines.

## Build check

`npm install` and `npm run build` completed successfully before packaging.
