# Boatshed Marketplace — Premium Design System Pass

This build intentionally avoids the previous cartoon/SVG hero direction.

## Design direction
- Photo-led marine brokerage feel.
- Dark cinematic hero with a real marina/yacht image treatment.
- Restrained Boatshed navy/off-white/orange palette.
- Editorial serif headings paired with clean system UI text.
- Minimal professional line icons used only as UI support.
- Fewer, larger sections with more whitespace.
- Premium listing cards instead of blocky Bootstrap cards.

## Image strategy
The homepage uses remote Unsplash marine photography as development placeholders. Replace these with licensed Boatshed/broker images before public launch.

## Icon strategy
The build includes small inline SVG line icons in `src/components/PremiumIcons.tsx`. They are deliberately understated and not used as cartoon hero artwork.
